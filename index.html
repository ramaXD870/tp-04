Tablas

1. ¿Cuál es la etiqueta principal utilizada para crear una tabla en HTML?

Las tablas html se dicen con la etiqueta "table"que permiten a los desarrolladores web organizar datos como texto, imágenes y enlaces en filas y columnas.

2. ¿Qué etiqueta se utiliza para definir una fila en una tabla?

El elemento HTML "tr" define una fila de celdas en una tabla. Luego, las celdas de la fila se pueden establecerusando una combinación de elementos.

3. ¿Cuál es la etiqueta utilizada para crear una celda de encabezado en una tabla?

para esta etiqueta vamos a utilizar la etiqueta "th"

4. ¿Cuál es la etiqueta utilizada para crear una celda de encabezado en una tabla?

para esta etiqueta utilizaremos las letras "td"

5. ¿Qué atributo se utiliza para especificar cuántas columnas debe abarcar una celda?

esta etiqueta usaremos la frase "colspan"a la vez que ponemos un numero

6.¿Qué atributo se utiliza para especificar cuántas filas debe abarcar una celda?

para esta etiqueta se usa "rowspan"

7. Si necesitamos crear una tabla con 10 filas ¿qué etiqueta y cuántas veces se debe utilizar para definir las filas?

se utilizan 10 "tr" para crear una tabla que contenga 10 filas.
